<HistoryGnuArmEclipse date="February 2018"/>

The initial name of the package was
[GNU MCU Eclipse ARM Embedded GCC](https://github.com/ilg-archived/openocd)
and was first published on `npmjs.com` as
[`@gnu-mcu-eclipse/arm-none-eabi-gcc`](https://www.npmjs.com/package/@gnu-mcu-eclipse/arm-none-eabi-gcc).

### March 2018

The first release of **GNU MCU Eclipse ARM Embedded GCC** was
[v6.3.1-1.1](/blog/2018/03/31/arm-none-eabi-gcc-v6-3-1-1-1-released/).

### July 2019

<HistoryRenameXpack/> and the npm scope was changed from `@gnu-mcu-eclipse` to
[`@xpack-dev-tools`](https://www.npmjs.com/package/@xpack-dev-tools/arm-none-eabi-gcc)
(in [v7.3.1-1.2](/blog/2019/07/26/arm-none-eabi-gcc-v7-3-1-1-2-released/)).

### July 2020

<HistoryArmLinuxAdded/> (in [v9.3.1-1.1](/blog/2020/07/03/arm-none-eabi-gcc-v9-3-1-1-1-released/)).

### October 2020

After extensive experimentation, support for Python
scripting was added to GDB, using an embedded Python 3 instance
(in [v9.3.1-1.4](/blog/2020/10/23/arm-none-eabi-gcc-v9-3-1-1-4-released/)).

### October 2021

<HistoryWindowsUcrt/> (in [v10.3.1-2.1](/blog/2021/10/25/arm-none-eabi-gcc-v10-3-1-2-1-released/)).

### December 2021

<HistoryAppleSiliconAdded/> (in [v10.3.1-2.3](/blog/2021/12/07/arm-none-eabi-gcc-v10-3-1-2-3-released/)).
